Inventory Management System

A Java-based Inventory Management System built with OOP principles and JSON persistence.
The application allows users to manage products, stock quantities, categories, and inventory operations through a CLI interface.

Features
Add products
Remove products
View all products
Increase stock
Decrease stock
Search products by name
Filter products by category
JSON data persistence
Automatic save/load system

How to Run
Open the project in your Java IDE
Run Main.java
Use the CLI menu to manage inventory
