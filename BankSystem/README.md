Java Banking System (CLI Project)

A simple but well-structured Java-based banking system built using 
Object-Oriented Programming principles, layered architecture, and file-based JSON persistence.

This project simulates core banking operations in a command-line interface.

Features
Create bank accounts
Deposit money
Withdraw money
Transfer between accounts
View account details
View transaction history
Persistent storage (data saved in JSON file)

Data Persistence
Accounts are stored in a local accounts.json file
Data is automatically saved after every operation
Data is loaded when the application starts

Technologies Used
Java 17+
Object-Oriented Programming (OOP)
Collections Framework (HashMap, List)
File Handling
Jackson (JSON serialization)

How to Run
1. Clone the repository
git clone https://github.com/your-username/java-banking-system.git

2. Open in your IDE (IntelliJ / Eclipse)
3. Run Main.java
